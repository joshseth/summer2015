Summer 2015 Research Goals and Ideas

Josh Schiffman

  1. Check for bugs

    Why is the simulation terminated after approximately 68,000 generations?

  2. ``Discreteness" of fitness

    Use instead of ; take a function of the Hamming distances between TFs and TFBS. Use for matrix entries, where is the number of differences.

  3. Allow generation overlap

    Strategy: Currently at the start of every new generation all individuals in the previous generation die. To simulate overlapping generations, only a fraction of individuals are selected to die every generation (should this be proportional to fitness?). Note: Paul Thomas thinks that drops in fitness could be due to the current simulation never competing offspring against potentially more fit parents.

  4. Snowball Mechanism

    Incompatibility vs. Time is not an adequate measure of the Snowball effect. We are really interested in whether or not the number of possible DMI mutations go up quadratically with the number of changes. Therefore, we should look at the number of fixed mutants that cause death when placed in an ancestor as a function of the number of fixed mutations. This relationship should be exponential.

    Strategy: (1) run simulation until population reaches maximum fitness and save as ``ancestral population." (2) Continue simulation for X generations and search for fixed mutations every generation. Place fixed mutations in (an) ancestor(s) and calculate fitness/incompatibility. (3) Plot ancestral incompatibility as a function of fixed mutations.

  5. Quantify the degree of constraint.

    (1) nucleotide: the number of substitutions vs. the mutation rate (i.e. expected without selection). (2) network(G): Euclidean distance vs. expected under just mutation.

  6. Check if Incompatibility can be due to just sequence drift without rewiring.

    Strategy: If Incompatibility is due solely to sequence drift and not GRN rewiring, we would observe Incompatibility only when genome sequences are hybridized and not when GRNs are hybridized. Check if switching matrix entries does not cause Incompatibility in cases where switching sequences does.

  7. Pleiotropy

    How does selection act (if at all) on pleiotropy; specifically, TFBS motif pleiotropy? i.e. are matching ``nucleotide'' slots shared across multiple regulatory regions? In a GRN if both and have positive elements, are these scores due to the same or different matches in the same TF/TFBS, significantly?

    Strategy: (1) mutate nucleotides, (2) record the GRN-level impact of each sequence change on each GRN entry (if a given mutation alters more than one matrix entry it is ``motif pleiotropic," and (3) find the covariance of changes between matrix elements to (4) determine if motif pleiotropy is under negative or positive selection.

  8. DSD rate estimate and a critical migration value that entails speciation.

    Strategy: (1) Add a geographic component to the model. Individuals are assigned a location on a line or grid and their offspring migrate to other locations with a probability proportional to distance. Mates are also chosen with a probability proportional to distance. (2) Run simulation for many generation and plot Incompatibility vs. pairwise distance of all organisms in a population.

  9. Extending the regulatory region length.

    Would we see more mutations of small effect?

  10. Is modularity selected for?

    See if ``modularity" goes up as a population size increases or down as selection decreases.

  11. Predict in vivo DSD rates.
